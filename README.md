# EMC Shop

EMC Shop for EMC Discord Server

## Prerequisites

* MariaDB (MySQL) Database
* Meilisearch
* Cloudflare R2
* Discord Application
* Stripe (Stripe Connectも必要)

## Setup

1. `.env.sample`を`.env`にコピー
2. 値を入力
3. `pnpm i`
4. `pnpm run build`
5. `pnpm run start`
